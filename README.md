# Classical guitar sheet music

[![PDF Generation Status](https://github.com/pinbe/partitions/actions/workflows/build-score.yml/badge.svg)](https://github.com/pinbe/partitions/actions)

In this repository, you'll find the classical guitar sheet music I play in my free time.

To play them now, you can visit the score index at https://pinbe.github.io/partitions/ and click on the links to download up to date PDF scores or MIDI files.

The scores are generated from LilyPond source files, which are available in the repository. If you want to edit the scores, you can clone the repository and edit the .ly files. For this, you may use the [LilyPond software](https://lilypond.org/), with or without the great editor [Frescobaldi](https://frescobaldi.org/).

Almost all scores include a utility file ([guitar-helpers.ly](https://github.com/pinbe/partitions/blob/master/commons/guitar-helpers.ly)) that provides what is useful for guitar-specific notation. Pay attention that file is correctly included in the .ly or edit the include statement accordingly.
